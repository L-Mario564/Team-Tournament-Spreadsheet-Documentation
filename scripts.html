Spreadsheet Scripts

A description of each script within the Spreadsheet Scripts menu.

Config

Script Name Description
Authorize Scripts Script that authorizes the scripts. In other words, it's the script that allows you to run all other scripts.
Save An API Key It stores the inputted API key into Google Sheet's Property Service, the API key is used to get data for beatmaps and MP links. If you've already inputted one and wish to change it, run this script again to change your key.

Create

Script Name Description
Create New Scores Sheet Creates a copy of the Scores Template sheet and uses the Settings Sheet and Mappools Sheet as references for it's set up.
Create New Scouting Sheet Creates a copy of the Scouting Template sheet and uses the Settings Sheet and Mappools Sheet as references for it's set up.
Create New Availability Sheet Creates a copy of the Availability Template sheet and uses the Settings Sheet as reference for it's set up.

Update

Script Name Description
Update Settings Updates every sheet to match the inputs provided in the Settings Sheet.
Update Mappools Updates the Mappools Sheet to match the mappools found in the tournament's Main Sheet.

Delete

Script Name Description
Delete Settings Deletes all the inputs in the Settings Sheet.
Delete Mappools Deletes all the mappools in the Mappools Sheet. This script doesn't remove mappools from Properties Service.
Delete API Key Deletes the API key from the spreadsheet if there is one.
